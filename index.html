Preparing your code for your website

Before you start writing your code you should write a declaration to let web browsers know that we are using HTML. The declaration should look like this:

<!DOCTYPE html>

The declaration provides the browser with two pieces of information (the type of document and the HTML version to expect), but it doesnâ€™t actually add any HTML structure or content. To create HTML structure and content, we must add opening and closing <html> tags after declaring <!DOCTYPE html>:



<!DOCTYPE html>
<html>

</html>
