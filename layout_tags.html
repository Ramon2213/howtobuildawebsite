HTML layout tags



tag meaning example
<h1>,<h2>,<h3>,<h4>,<h5>,<h6> The <h1> to <h6> tags represent six levels of section headings. <h1> is the highest section level and <h6> is the lowest. <h1>This is a heading tag</h1>
<p> it is the most common tag for larger chunks of text. <p>This is a paragraph tag</p>
<a> used to make a link to another location, this could be on the same page or a new one, activated by clicking on the text or image <a>This is an anchor tag</a>
<button> is an interactive element activated by clicking on it. Once activated, it then performs a programmable action, such as submitting a form or opening a dialog. To use this to open a link you can put the anchor tag within the button tag <button>This is an button tag</button>
